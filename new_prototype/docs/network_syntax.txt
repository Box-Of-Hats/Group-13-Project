Syntax for commands sent via the network. First 3 characters of a message string indicate the command to be called, and are listed below. 

spd: setPredecessor
uft: updateFingerTable(s, i)
cpf: closestPreceedingFinger(int key)

// fpd: findPredecessor(int id)
// fsu: findSuccessor(int id);
	Takes the guid of the node to find the successor of and returns the FingeredNode.
